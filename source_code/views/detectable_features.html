Chart Checker - All detectable misleading features

Name Description How is this misleading Example
Inconsistent axis scales Inconsistent axis scales can make certain data points appear disproportionately large or small compared to others. This can exaggerate or downplay differences between data points. This can be misleading as it distorts the representation of data, making some sections of the data appear disproportionately large or small compared to others. This misrepresentation can lead to incorrect interpretations and conclusions.
Inconsistent placement of tick labels Tick labels are used to indicate the scale of a chart. If they are placed with inconsistent spacing, it can be hard to identify the scale in certain parts of the chart. This can be misleading as it highlights sections of data that contain more tick labels while making it harder to judge the value of data points in other sections with less tick labels.
Inverted y-axis An inverted y-axis in a chart means that the values on the vertical axis are shown in reverse order, with the higher values at the bottom and the lower values at the top. This can be misleading because it flips the expected progression of the data, making upwards trends appear like downwards trends and visa versa. This can cause confusion and misinterpretation of the data.
Misleading aspect ratio A chart with a misleading aspect ratio distorts the proportions and relationships between data points by choosing scales for the x-axis and/or y-axis that are not optimal for the displayed data. This can be misleading as it makes trends across the data appear more or less steep then they actually are. This misrepresentation can lead to incorrect interpretations and conclusions.
Missing labels Labels in a chart provide context and clarity about the data being presented. If they are missing, this information is not available. Such labels include the chart title, titles for the x-axis and y-axis or a legend. This is misleading as, without labels, we may lack critical context about the presented data, which makes it difficult to interpret values and make accurate comparissons.
Mutiple Axis A chart that contains multiple data series can feature multiple x-axis or y-axis, which can represent different scales or measurements. This can be misleading, as it can become unclear which axis belongs to which data series. Furthermore different scales or measurements on each axis can distort comparisons and show misleading patterns. This can lead to incorrect conclusions about relations between different data series.
Truncated y-axis A truncated y-axis removes a portion of the vertical axis scale, highlighing only a certain section of the data. This can be misleading as it makes the displayed data look more extreme or exaggerated than it actually is which can lead to misinterpretation and false impressions
